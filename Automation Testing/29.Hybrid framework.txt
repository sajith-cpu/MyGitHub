Explain About your framework:

In our project we are using hybrid framework. Hybrid framework is combination of all the diff framework in selenium.
In this framework we are designing one test script in the format of one method by preparing business operations using page object model and performing validation using reusable method, and we are passing the data from excel sheet.

We are storing the output in the format of screenshot. Once we complete all the script designing part we are adding the classes to Regression.xml or SomkeTesting.xml file. We will execute the .xml file and after execution xml  we will analyze the result and check the TestNG report and if any test scenario is failing we will debug and contact with development team.
